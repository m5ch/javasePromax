# XML语法规则

## 创建规则

1. 后缀为 .xml

2. 文档申明必须是第一行

## 标签(元素)规则

1. 标签由一对尖括号和合法标识符组成:
   
        <name></name>，
   **必须存在一个根标签，有且只能有一个。**
   
2. 标签必须成对出现，有开始，有结束:
   
        <name></name>
   
3. 特殊的标签可以不成对，但是必须有结束标记，如:
   
        <br/>
4. 标签中可以定义属性，属性和标签名空格隔开,属性值必须用引号引起来
   
        <student id = “1" ></name>
   
5. 标签需要正确的嵌套

## 其他组成

1. XML文件可以定义注释信息

        <!--注释内容-->
2. 可以存在以下特殊字符

        &lt;    <   小于
        &gt;    >   大于
        &amp    &   和
        &apos   '   单引号
        &quot   "   双引号
3. 可以存在CDATA区

        <![CDATA[内容]]>
